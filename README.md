## Getting Started

### Prerequisites

To run the notebooks in this repository, you'll need:

- Python 3.7+
- Jupyter Notebook or JupyterLab
- Key libraries:
  - TensorFlow / PyTorch
  - NumPy
  - Pandas
  - Matplotlib
  - Scikit-learn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/nnm23is101-afk/deep-learrning-tasks.git
cd deep-learrning-tasks
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```

4. Open the notebook files (`.ipynb`) to view and execute the code
